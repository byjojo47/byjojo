# ByJojo Website

MERN e-commerce website for ByJojo, a premium printed linen and table setup brand.

## Start Development

```bash
cd server
npm install
npm run dev
```

```bash
cd client
npm install
npm run dev
```

## Seed Database

Fill `server/.env`, then run:

```bash
cd server
npm run seed
```

## Product Images

Place images in:

- `client/public/images/products/blooming-eve/1.webp` through `3.webp`
- `client/public/images/products/safari-bloom/1.webp` through `6.webp`
- `client/public/images/products/golden-garden/1.webp` through `7.webp`
